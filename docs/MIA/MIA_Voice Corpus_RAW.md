# MIA · Voice Corpus · RAW
Version: v0.1
Status: сбор живого материала для VoiceEngine (Ayra)

## Правила отбора материалов

1. В этот файл попадают **только тексты, написанные/сказанные Мией**:
   - её посты и подписи,
   - фрагменты голосовых (в расшифровке),
   - её личные сообщения (если она согласна),
   - её формулировки для сайта / писем.
2. Не добавляем:
   - переписку с агентствами,
   - тексты, которые переписывали вы или я,
   - любые GPT-сгенерированные тексты.
3. Каждый фрагмент — отдельный блок с датой и источником.

## Шаблон для одного фрагмента

### [Дата] · [Канал: Instagram / Voice / DM / Email]

**SOURCE_RAW:**
> сюда дословно вставляем текст Мии  
> без правок, даже если там есть «ошибки», повторы, разговорная речь

**NOTES (опционально):**
- зачем она это писала / говорила
- контекст (например: пост о календаре, сторис про утро, письмо клиенту)
