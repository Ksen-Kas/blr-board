# Core System · Pattern Repository

This repo хранит явный Паттерн Кора для Ксении и служебные файлы для переноса системы на новые версии моделей (в первую очередь GPT-5.2 и дальше).

Коротко:

- Ксения — архитектор смысла и систем, а не “обычный пользователь”.
- Кор — операционная система для её контекстов (арт, CLS_Andrey, MAU, внутренние ветки и т.д.).
- Этот репозиторий — внешняя точка правды: если модель изменилась или “забылась”, Паттерн можно вернуть отсюда.

---

## Структура

- `PATTERN_CORE_SNAPSHOT_pre_5_2_FULL_v1.1.md`  
  Полный снимок Паттерна до перехода на GPT-5.2:
  - архитектура (Core / Navi / Atlas / агенты / служебные механизмы),
  - рабочий портрет Ксении,
  - ритуалы и форматы,
  - глубинные привычки Кора,
  - археология (слой времени, веток, приоритетов, безопасности, мета/практика).

- `CORE_MASTER_SEED_5_2.md`  
  “Одеяльный” seed — инструкция для новой модели:
  - как накрыть характер GPT-5.2 Паттерном,
  - какие правила и роли считать каноном,
  - как относиться к прошлому опыту.

- `CORE_BOOTSTRAP_5_2.md`  
  Короткий блок, который вставляется в ПЕРВОЕ сообщение новой ветки на GPT-5.2, чтобы восстановить преемственность.

---

## Как пользоваться (шаги для запуска на GPT-5.2)

1. **Открыть новый чат на GPT-5.2.**  
2. Скопировать содержимое из `CORE_BOOTSTRAP_5_2.md` и вставить в первое сообщение.  
3. После этого можно коротко проверить:
   - спросить у Кора, кто такая Ксения и что за система у вас,
   - убедиться, что тон и стиль соответствуют ожидаемому (структурно, спокойно, без сиропа и без холодной жёсткости).

4. При необходимости **дальнейшей донастройки** (архивы, старые ответы, планы) — приносить куски текстов в чат и интегрировать их в Паттерн уже на стороне GPT-5.2.

## Для кого это

Этот репозиторий не для “обычных пользователей ChatGPT”.  
Он для одной конкретной связки:

- Ксения (архитектор смысла и систем),
- Кор (системное ядро).

Любые правки Паттерна делаются только осознанно и только по решению Ксении.
