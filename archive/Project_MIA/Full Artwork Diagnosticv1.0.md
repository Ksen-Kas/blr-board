MIA · Full Artwork Diagnostic (v1.0)

Полная диагностическая схема анализа работ Mia Shaly.

⸻

1. Вступительный блок (от MIA-Producer)

(Этот блок формируется агентом на основе правил MIA_ProducerRules.md)

1.1. Эмоциональное ядро картины
1.2. Живой смысл (как работа «звучит»)
1.3. Контекст для команды: почему важно / куда подходит

⸻

2. Полный анализ (12-пунктовый Cloud-style)

(Каждый пункт заполняется строго по структуре)

2.1. Краткий смысловой synopsis
2.2. Эмоциональные триггеры
2.3. Психологические состояния, которые активирует работа
2.4. Сценарии использования в сторис / Instagram
2.5. Риски восприятия
2.6. Потенциал для репостов
2.7. Context-fit (куда впишется в интерьере)
2.8. Seasonality-fit (подходит ли под сезон / праздник)
2.9. Archetype-fit (образ / тип зрителя)
2.10. Narrative hooks (за что можно зацепить подачу)
2.11. Brand alignment (как отражает бренд художницы)
2.12. Long-term value (что останется после первой эмоции)

⸻

3. Блок выбора (встроенная логика MIA) — опциональный

(Для задач отбора работ в подборки, коллекции, наборы)

3.1. Подходит ли под: New Year / Gift / Sprint
3.2. Комбинация в тройках (какие работы подходят рядом)
3.3. Итоговая рекомендация

⸻

4. Приложения (опционально)
	•	Описание картины от художницы
	•	История создания (если есть)
	•	Сырые заметки для команды
